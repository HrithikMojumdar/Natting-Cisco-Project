### Firstly Static Routing ###

Router 1:
---------

en
conf t
int fa0/0
ip address 20.0.0.1 255.255.255.252
no shut
int fa0/1
ip address 10.0.0.1 255.0.0.0
no shut
ip route 30.0.0.0 255.0.0.0 20.0.0.2



Router 2:
---------

en
conf t

no shut
exit

ip address 30.0.0.1 255.0.0.0
no shut
exit
ip route 10.0.0.0 255.0.0.0 20.0.0.1
